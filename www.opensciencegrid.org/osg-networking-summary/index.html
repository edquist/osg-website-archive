OSG Networking Summary

OSG has a goal of support its members and affiliates to better understand, diagnose and utilize their networks.   Here is the summary of what OSG is  providing:

  • OSG will help sites and VOs deploy network measurement points based upon perfSONAR
  • OSG provides tools to manage, maintain and monitor your network links
    • OSG hosts an OIM registration service for perfSONAR instances
    • Currently OSG provides a perfSONAR mesh-configuration GUI via OIM which allows groups of perfSONAR instances to use centrally configured tests.
    • OSG has developed a standalone perfSONAR mesh-configuration tool that will be released with the perfSONAR v4.0 Toolkit as an OSG contribution.
    • OSG runs an OMD/Check_mk service monitoring and alerting system which tracks the status of the various perfSONAR toolkit services at all OSG sites
    • OSG runs a MaDDash instance which graphically displays the status of various network metrics, summarizing test results.
  • OSG hosts a long-term network metric datastore to view historical network data for research or historical investigations
  • OSG is developing network alarming and alerting tools to identify network problems and enable its users to register for alerts that may be important to them.
